# College To-Do List

A simple and user-friendly To-Do List application designed for students to organize daily tasks, assignments, and academic activities.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Clean and responsive user interface
* Real-time task management

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Use

1. Enter a task in the input field.
2. Click the Add button to create a new task.
3. Mark tasks as completed when finished.
4. Delete tasks that are no longer needed.

## Learning Outcomes

* DOM Manipulation
* Event Handling
* Dynamic Element Creation
* JavaScript Functions
* Frontend Development Fundamentals

This project was developed to improve JavaScript and web development skills while creating a practical productivity tool for students.
